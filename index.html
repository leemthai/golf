Golf Assets

Static asset repository (GLTF, textures, etc.) served via GitHub Pages.

Usage

Base URL:

https://YOURUSERNAME.github.io/golf-assets/

Example:

/models/hole_01.glb

Models

  • hole_01.glb

Textures

  • grass.png

Notes

  • Keep paths stable (URLs are used externally)
  • Prefer versioned folders (e.g. /v1/) for breaking changes
  • Large files: watch GitHub size limits (~100MB/file)
